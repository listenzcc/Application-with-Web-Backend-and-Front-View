# 搭建仿真动态显示系统说明书

## 1. 系统目标

集成：

- 传感器实时数据读取
- FDS/HYSPLIT 泄漏扩散仿真
- 地图/室内风险动态渲染
- 实时与历史会话切换

## 2. 代码模块概览

- `python/main.py`：主 UI + 路由 + 仿真入口
- `fds/simulate.py`：FDS 仿真核心
- `hysplit/simulate.py`：HYSPLIT 仿真核心
- `sensors/sensor_reader.py`, `sensors/sensor_writer.py`, `sensors/ui.py`
- `explorer/toxic_gas.py`：气体数据库管理
- `auth/`：权限认证与用户控制
- `components/layout.py`：主导航布局
- `static/html/map.html`, `static/html/room.html`：前端地图/场景模板

## 3. 仿真调用流程

### FDS

1. UI: `/simulationFDS` 点击“开始 FDS 模拟”
2. 传感器数据：`SensorDataReader().get_sensor_info()` + `get_latest_data`
3. 执行：`simulate_with_fds(sensors)` -> 返回 `session`
4. 显示：
   - 更新会话历史 `get_fds_simulation_result_history()`
   - `update_room(session)`
   - 通过 `iframe src="/room?session=xxx"`

### HYSPLIT

1. UI: `/simulationHysplit` 点击“开始 hysplit 模拟”
2. 读传感器
3. SESSION：`mk_hysplit_session()`
4. 运行：`simulate_with_hysplit(sensors, session)`
5. 显示：同 FDS，`/map?lat=..&lon=..&zoom=..&session=..`

## 4. 地图动态显示

### `/map` 入口

- 读取 `static/html/map.html`
- 替换变量：`{{zoom}}`, `{{lat}}`, `{{lon}}`, `{{session}}`
- 返回 `HTMLResponse`

### `/room` 入口

- 读取 `static/html/room.html`
- 替换变量：`{{session}}`

### iframe 嵌入

- FDS/Hysplit 页面 `map_card` 部分：

```html
<iframe id="map-iframe" src="/map" ...>
```

- JS 动态修改 `src` 为 `/map?...` 或 `/room?...`

## 5. 仿真结果获取 API

- `GET /get_fds_simulation_result/{session}` -> JSON {files:[...]}
- `GET /get_fds_simulation_frame?session=&frame=` -> PNG image
- `GET /get_hysplit_simulation_result/{session}` -> JSON {files:[...]}
- `GET /get_hysplit_simulation_frame?session=&frame=` -> PNG
- `GET /get_hysplit_simulation_table_json?session=` -> table JSON

## 6. 文件结构说明

- `fds/simulation/<session>/...`
- `hysplit/simulation/<session>/...`
- `data/case/...`：案例数据源
- `static/html/...`：前端可视化模板
- `conf/project.yml`：项目元数据（名称、抽象）

## 7. 部署建议

- 使用 `uvicorn` 生产启动（去掉 reload）：

```bash
uvicorn python.main:app --host 0.0.0.0 --port 8000 --workers 4
```

- 数据目录：使用独立存储卷挂载 `fds/`, `hysplit/`, `db/`, `data/`
- 定期清理历史仿真与传感器数据（脚本可补充）
- 生产强制 HTTPS + 权限策略加固

## 8. 关键点校验

- `auth` 登录后 `app.storage.user['authenticated']=True`
- `EnhancedAuthMiddleware` 保护非白名单页面
- `session_manager` 管理会话 `session_id` cookie
- `map`/`room` : session 与可视化同步
- `simulate_with_fds` / `simulate_with_hysplit` 为核心仿真计算模块

## 9. 拓展建议

1. 增加 WebSocket 实时帧推送
2. 引入 `scheduler` 定时监测传感器动态触发仿真
3. 进度指示（长时间仿真）
4. 异常处理 + 仿真失败回滚
5. 前端图层在线开关（泄漏浓度、风场、管廊）
