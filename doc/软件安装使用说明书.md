# 软件安装使用说明书

## 1. 概述

本项目：`Application-with-Web-Backend-and-Front-View`，基于 `NiceGUI` + `FastAPI`，实现：

- 传感器数据展示（`/sensors`）
- 气体管理与数据库（`/gasExplorer`）
- 事故案例教育（`/accidents`）
- 案例浏览（`/caseBrowser`）
- FDS/HYSPLIT仿真界面（`/simulationFDS`, `/simulationHysplit`）
- 地图动态播放接口（`/map`, `/room`）
- 权限认证与用户管理（`/login`, `/privilege`）

## 2. 软件环境

- 操作系统：

  - Windows（推荐）

    - Windows 11 专业版
    - 版本号 25H2
    - 安装日期 2025/2/15
    - 操作系统版本 26200.8037
    - 体验 Windows 功能体验包 1000.26100.300.0
  - Linux（可选）

    - 发行版：Ubuntu 22.04.5 LTS
- Python：>=3.11（项目脚本 `serve-forever.ps1` 激活 `conda python3.11`）

- 推荐：Anaconda/Miniconda

## 3. 依赖安装

当前依赖在 `python/requirements.txt`：

- pandas

## 4. 数据库初始化

项目数据库脚本位于：

- `python/init_auth_db.py`
- `python/init_gas_db.py`
- `python/init_sensors_db.py`

执行：

powershell

Copy

```powershell

```

另外包含清空/重置脚本：

- `clear-fds-simulations.ps1`
- `keep-writing-sensors.ps1`

## 6. 访问入口

- 首页：`/`
- 登录：`/login`
- 事故案例：`/accidents`
- 气体库：`/gasExplorer`
- 案例库：`/caseBrowser`
- 传感器：`/sensors`
- 用户/权限：`/privilege`
- FDS 仿真：`/simulationFDS`
- HYSPLIT 仿真：`/simulationHysplit`
- 资源接口：`/map`, `/room`, `/data/case/{case_name}`

## 7. 维护与升级

- 修改后重启服务
- 退出：`Ctrl+C`
- 日志：可加 `uvicorn` 参数或 NiceGUI 输出
- 清理：
  - 删除 `db/*`
  - 删除 `fds/simulation/*`, `hysplit/simulation/*`
