# 软件操作指南

## 1. 登录与权限

- 访问 `/login`
- 默认内置角色（`testuser`、`guest`）在 `main.py` 中创建（首次运行）
- 权限由 `auth/` 模块管理：
  - `user_service`
  - `PermissionManager`
  - `AuthContext`

## 2. 全局导航（`components/layout.py`）

主导航链接：

- “扩散仿真(hysplit)” -> `/simulationHysplit`
- “仿真案例” -> `/caseBrowser`
- “传感器监控” -> `/sensors`
- “气体数据库” -> `/gasExplorer`
- “事故教育” -> `/accidents`
- “个人资料” -> `/profile`
- “权限管理” -> `/privilege`

## 3. 首页 `/`

- 未登录显示：“立即登录”“了解更多”
- 已登录显示跳转按钮：案例库、传感器监控、气体数据库
- 读取 `conf/project.yml` `abstract` 作为说明文本

## 4. 案例库 `/caseBrowser`

- 加载 `data/case` 下所有子目录
- 目录树浏览、文件预览、文件下载
- 支持图片、PDF、文本预览
- 支持返回上级目录、文件类型图标

## 5. 气体数据库 `/gasExplorer`

- `GasManagementUI`
- 创建/添加/删除气体
- 查询、排序、分页、统计（总量、毒性分布、分子量/沸点范围）
- `permission_manager` 控制“编辑/删除”按钮
- 导出 Excel（`gas_db.export_to_excel`）

## 6. 事故教育 `/accidents`

- 加载 `data/accidents/a-20251221.xlsx`
- 按“案例场景分类”筛选
- 关键词搜索（案例名称、泄漏气体、位置、概要）
- 详情对话框、导出事故信息为 TXT

## 7. 传感器监控 `/sensors`

- 依赖组件：
  - `sensors/sensor_reader.py`
  - `sensors/sensor_writer.py`
  - `sensors/ui.py`
- `SensorDataReader().get_sensor_info()`
- `latest_sensor_data` 接口：`/latest_sensor_data`

## 8. 用户权限 `/privilege`

- 查看当前用户信息
- 查看用户列表、编辑、删除（权限 `view_users/create_user`）
- 新建用户（`guest/user`角色）
- 注销：`/profile` -> 按钮

## 9. 仿真操作

### FDS `/simulationFDS`

- 点击“开始 FDS 模拟”
- 读传感器数据（`SensorDataReader`）
- 调用 `simulate_with_fds(sensors)`
- 更新 `<iframe id="map-iframe"> /room?session=...`
- 读取历史：`get_fds_simulation_result_history()`
- 图片/数据查询接口：
  - `/get_fds_simulation_result/{session}`
  - `/get_fds_simulation_frame?session=&frame=`

### HYSPLIT `/simulationHysplit`

- 点击“开始 hysplit 模拟”
- `mk_hysplit_session()` + `simulate_with_hysplit(sensors, session)`
- 更新地图 `update_map(session)`
- 历史列表：`get_hysplit_simulation_result_history()`
- 接口：
  - `/get_hysplit_simulation_result/{session}`
  - `/get_hysplit_simulation_frame?session=&frame=`
  - `/get_hysplit_simulation_table_json?session=`

## 10. 调试与日志

- `app.add_middleware` 命中 `EnhancedAuthMiddleware`
- 错误 404 处理 `exception_handler_404`
- 可通过浏览器 DevTool 查 `console` 和网络请求

## 11. 安全与退出

- 登出：`session_manager.remove_session` + `app.storage.user.clear()`
- 入口：`/profile` “logout”
- 生产中建议开启 HTTPS、`Secret` 并禁用 `reload=True`
